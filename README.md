## 🛠 Troubleshooting common issues

If you face problems, use these steps to fix your connection or performance.

**The game does not launch.** 
Close all running Roblox processes through the Task Manager. Restart WindowsDelta. If the error persists, uninstall the client and install it again.

**The menu screen looks blurry.** 
Check your monitor resolution in the Windows display settings. Ensure that your scaling is set to 100%.

**Performance remains the same.** 
Open the client settings menu. Ensure that the performance mode toggle is on. Check that your computer uses your dedicated graphics card instead of the integrated chip.

**The client crashes on startup.** 
Ensure your graphics card drivers are current. You can update these via the website of your hardware maker.
